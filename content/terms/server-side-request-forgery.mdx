# Server-Side Request Forgery

OWASP describes Server-Side Request Forgery (SSRF) as a vulnerability that allows an attacker to induce a server to make HTTP requests to an arbitrary domain, which can expose internal systems or sensitive data.

Source: [OWASP Top 10 - Server-Side Request Forgery](https://owasp.org/Top10/A10_2021-Server-Side_Request_Forgery_%28SSRF%29/)
