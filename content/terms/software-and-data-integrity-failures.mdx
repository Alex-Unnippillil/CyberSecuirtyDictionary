OWASP states that Software and Data Integrity Failures occur when code and infrastructure do not protect against integrity violations, potentially allowing malicious updates or unauthorized changes.

Source: [OWASP Top 10 - Software and Data Integrity Failures](https://owasp.org/Top10/A08_2021-Software_and_Data_Integrity_Failures/)
